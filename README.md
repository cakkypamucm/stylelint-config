This is `stylelint` config (`scss` + `standard-scss` + `idiomatic-order` + `prettier`)

### Install

```shell
npm i --save-dev github:cakkypamucm/stylelint-config
```

### Use

Add `stylelint-config-cakkypamucm` to the "extends" array in your `stylelint` config file

<pre>
{
    "extends": ["some-other-config-you-use", <b>"stylelint-config-cakkypamucm"</b>]
}
</pre>

### For synergy

Add [eslint-config-cakkypamucm](https://github.com/cakkypamucm/eslint-config) to your project
